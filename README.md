# 📊 Sales Performance Analysis Dashboard

## 📌 Project Overview

This project presents a comprehensive **Sales Performance Analysis Dashboard** built using **Power BI**.
The dashboard helps analyze sales data across different **cities, products, promotions, and time periods** to identify business trends and performance insights.

The goal of this project is to transform raw transactional sales data into **interactive visual insights** that help understand product performance, customer demand, promotional effectiveness, and sales growth patterns.

---

## 📂 Dataset Description

The dataset contains transactional sales records with the following key fields:

* Order ID
* Customer ID
* Product ID
* Promotion ID
* Order Date
* Discount Percentage
* Discount Value
* Total Sales
* Net Sales
* Profit
* Units Sold
* Price per Unit

This dataset allows analysis of **sales performance, product demand, and profitability**.

---

## 📊 Dashboard Features

### 🌍 Sales by City

A geographical map visualization shows the **distribution of sales across different Indian cities**, helping identify high-performing locations and regional demand patterns.

---

### 📦 Order Metrics

* **Total Orders:** 3510
  Provides a quick overview of overall business activity and order volume.

---

### 🎯 Promotion Analysis

The dashboard analyzes **average discount by promotion categories**, including:

* Weekend Flash Sale
* Clearance Sale
* Summer Sale
* New Year Special
* Festive Diwali

This helps evaluate **which promotional campaigns provide the highest discounts and drive customer engagement**.

---

### 💰 Profit vs Net Sales Analysis

A scatter plot visualizes the relationship between **profit and net sales**, helping understand how sales revenue translates into profit across transactions.

---

### 📈 Sales Trend Over Time

A line chart shows the **sales trend across multiple years**, allowing analysis of:

* Yearly sales growth
* Decline or growth periods
* Long-term sales performance
